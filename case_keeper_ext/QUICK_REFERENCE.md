## Search bar

| | |
|-|-|
| 🏠 | Cases home (Shift = float window, Ctrl/Cmd = new tab) |
| Type + Enter | Search for one or more case numbers |
| 🔍 | Search |
| ↻ | Refresh all vaulted cases |
| ⧉ | Copy all case details to clipboard |
| ▼ / ▲ | Show / hide both result sections |
| — | Hide entire toolbar |
| Green **+** | Restore hidden toolbar |
| Right-click or Ctrl+click input border | Hide toolbar |

---

## Viewing Case (auto-detected)

| | |
|-|-|
| 👁 click | Open follow popup |
| 👁 right-click | Copy follow URL |
| ☁️ click | Open in Salesforce console |
| ☁️ right-click | Copy Salesforce URL |
| ↻ | Refresh with latest API data |
| ★ / ☆ | Add to / already in Case Vault |
| ⧉ | Copy case details to clipboard |
| Case # click | Navigate to case |
| `url` | Copy plain URL |
| `md` | Copy Markdown link |
| + | Add / edit note |
| − | Delete note |
| ▶ | Expand to show subject / status / owner |
| ▼ | Collapse back to icons-only |
| — | Hide section |
| ▲ | Show section |
| ✕ | Dismiss (stop tracking this case) |

## Case Vault header

| | |
|-|-|
| ↓ | Export vault to JSON file |
| ↑ | Import cases from JSON file |
| ◀ | Collapse all cases |
| ▶ | Expand all cases |
| ✕ | Empty vault |
| — | Hide vault content |
| ▲ | Show vault content |

## Per-case controls (Case Vault)

| | |
|-|-|
| 👁 click | Open follow popup — green = following, dim = not |
| 👁 right-click | Copy follow URL |
| ☁️ click | Open in Salesforce console |
| ☁️ right-click | Copy Salesforce URL |
| ↻ | Refresh this case with latest API data |
| ⧉ | Copy case details (visible even when minimized) |
| Case # click | Navigate (current tab) |
| Case # Shift+click | Open in floating window |
| Case # Ctrl/Cmd+click | Open in new tab |
| Case # right-click | Open in new tab |
| `url` | Copy plain URL |
| `md` | Copy Markdown link |
| + | Add / edit note |
| − | Delete note |
| ◀ | Collapse case row |
| ▶ | Expand case row |
| ✕ | Remove from vault |
| ✕ right-click | Collapse instead of remove |
| Drag row | Reorder |
| Right-click row | Collapse that case |
| 👁 viewing badge | This case is currently open in Viewing Case |
